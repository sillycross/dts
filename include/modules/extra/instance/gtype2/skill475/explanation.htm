欢迎参加周五活动模式
以下是这个游戏模式的规则介绍
请认真阅读哦～ 		►
下一页
◄
上一页 		在这个模式中，你不会死亡。如果你因为任何原因死亡了，在15秒后你就会复活。
游戏将持续2禁，在游戏结束时，拥有最多胜利点数的玩家将会获胜。 		►
下一页
◄
上一页 		胜利点数可以通过击杀NPC或其他玩家获得。
击杀NPC获得的胜利点数是固定的，详见下表：
  
NPC种类胜利点数
小兵20
全息幻象250
真职人750
电波幽灵750
杏仁豆腐1200
黑幕 Acg_Xilin2500
 ►
下一页
◄
上一页 		而击杀玩家所获得的胜利点数是不固定的，与一个被称为赏金的参数有关。
每当玩家通过击杀NPC或其他玩家获得金钱时，
这名玩家的赏金就会增加，增加的数值与其获得的金钱数目相同。
注意如果击杀NPC或玩家后没有选择获得金钱，赏金是不会增加的。 		►
下一页
◄
上一页 		每当一名玩家通过任何方式击杀另一名玩家时，将会发生以下事情：
  • 击杀者将获得相当于15%被击杀者赏金的胜利点数。
  • 击杀者自身的赏金将增加相当于30%被击杀者赏金的数值。
  • 被击杀者的赏金将降低45%。
►
下一页
◄
上一页 		如果是通过战斗击杀玩家，还可以额外从以下选项中选择一项，作为对被击杀者的惩罚：
  • 销毁被击杀者30%的金钱（击杀者不获得）。
  • 获得被击杀者30%的金钱，但击杀者将因此失去同样数值的胜利点数。
  • 削弱被击杀者装备的武器12%的效果值。
  • 削弱被击杀者一件装备的防具20%的效果值。
  • 销毁被击杀者装备的饰品，或背包中的任何一件物品。

而通过陷阱等其他方式完成的击杀则只会获得胜利点数，无法选择上面的额外惩罚。 		►
下一页
◄
上一页 		需要注意的是，上述特殊规则只适用于击杀玩家。
对于击杀NPC，你依然可以获得其携带的全部金钱，或它的任何装备/物品。

如果你不幸被NPC击杀，或者遭遇各类意外事件而死亡，
前3次是不会有任何后果的（除了15秒的等待复活时间）。
但之后每次都会导致你的生命上限降低5%且失去3%的胜利点数。 		►
下一页
◄
上一页 		本模式中所有BOSS NPC（红暮/蓝凝/SCP/英灵）都没有进入战场，
因此开局不再有危险的不可以去的地点，但是躲藏也更加困难了呢。

商店中出售的陷阱价格相比正常模式也有不小的降低，要小心哦～
